My personal .vimrc
==================

Download
--------
In your home directory:
`$ git clone https://github.com/biskark/Vim-settings.git`

Installation
------------
`$ cp Vim-settings/.vimrc ~/.vimrc`

Notes
-----
This vimrc has setting for various plugins that may not apply to you if you
don't use those plugins.
